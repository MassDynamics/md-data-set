## Installation

To install md_data_set from GitHub repository, do:

```console
git clone git@github.com:MassDynamics/md_data_set.git
cd md_data_set
python -m pip install .
```

To install the package:

```console
python -m pip install .
```

To develop the package:

```console
python -m pip install -e '.[dev]'
pytest
```
